# Marketing

The launch story for `run-all-now` is benchmark-first.

## Assets

- `demo.tape`: VHS source for side-by-side terminal GIF.
- `launch-tweets.md`: short posts with benchmark hooks.
- `presentation.html`: five-page launch narrative for GitHub/social review.

## GIF rules

- Left pane: `npm-run-all`.
- Right pane: `run-all-now`.
- Show the exact command.
- Show wall time, CPU, and memory.
- Keep the final GIF under 20 seconds.
- Include raw benchmark scripts in the tweet thread.
