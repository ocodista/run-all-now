# Launch checklist

## Before public release

- [ ] Compatibility matrix has a clear MVP boundary.
- [ ] README includes real benchmark numbers and raw commands.
- [ ] `bench/` scripts run from a clean clone.
- [ ] `marketing/demo.gif` shows original left and native right.
- [ ] npm package installs on macOS arm64/x64 and Linux x64.
- [ ] GitHub issues are labeled and seeded with good first issues.
- [ ] Security review covers process/file inputs and shell escaping.
- [ ] Repository visibility flips from private to public.

## Launch copy

Position `run-all-now` as a fast native replacement for `npm-run-all`, not a rewrite for its own sake.

Lead with the measurable result: startup, throughput, memory, or a visible GIF.
